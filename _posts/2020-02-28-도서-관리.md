+ static class function

  + 클래스 변수

    + static String value=""

       엑셀에서 받아온 값이라 고정값으로 함

    + static String data배열;

       배열값을 공유해야 행의 데이터를 불러올 수 있어서

  + 클래스 매소드

    + public static void xcell()

       엑셀 행,열의 데이터를 읽어옴

    + public static String returnVal (String value)

       읽어온 데이터 값을 콘솔 출력, 리턴

    + public static void array (String value)

       받아온 데이터를 2차 배열에 넣음
